## ⚠️ Common Issues & Fixes

### "Firebase: Error (auth/unauthorized-domain)"
- **Solution:** Add your Vercel domain to Firebase authorized domains
- Take a screenshot of your Vercel domain (it's shown in Vercel dashboard)
- Go to Firebase Console → Auth → Settings → Authorized domains
- Copy-paste your domain and click Add

### CORS Errors ("blocked by CORS policy")
- **Solution:** Update `FRONTEND_URL` in Render to your actual Vercel URL
- Make sure you deploy after updating environment variable
- Test after 2-3 minutes for changes to take effect

### API Returns 404
- **Solution:** Verify `VITE_API_URL` in Vercel matches your Render backend URL
- Check it's exactly: `https://document-upload-1-hvgx.onrender.com`
- Redeploy frontend after checking

### "Network Error" on login
- **Solution:** Check both services are running:
  - Backend: https://document-upload-1-hvgx.onrender.com should show backend response
  - Frontend: Your Vercel URL should load the app
- Check browser console for exact error

### Vercel Build Fails
- **Solution:**
  - Check you selected `client` folder as root
  - Verify all environment variables are set
  - Check no `.env.local` is in git (it's in .gitignore)
  - Try rebuilding
